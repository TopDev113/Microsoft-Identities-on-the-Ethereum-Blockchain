Getting started with Metamask

  1. Open Firefox, or install it here.
  2. Install Metamask from the Firefox add-ins store.
  3. Read the Terms of Use and, if you agree, accept them.
  4. Use the Network dropdown in the top left to change your network to the Ropsten Test Net.
  5. Click the Metamask logo to open the add-in pane.
  6. Create a strong password for your Metamask install.
  7. Click the Account icon in the top right, and choose Import account.
